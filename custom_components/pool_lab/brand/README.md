# Brand Images

This directory contains the brand images for the Pool Lab integration.
Home Assistant 2026.3+ serves these automatically via `/api/brands/integration/pool_lab/`.

## Required files

| File | Purpose | Format | Size |
|------|---------|--------|------|
| `icon.png` | Square icon (integration list, device cards) | PNG, transparent background | 256x256 px |
| `icon@2x.png` | High-DPI square icon | PNG, transparent background | 512x512 px |
| `logo.png` | Rectangular logo (config page, detail views) | PNG, transparent background | 256x128 px |
| `logo@2x.png` | High-DPI rectangular logo | PNG, transparent background | 512x256 px |

## Optional dark mode variants

| File | Purpose |
|------|---------|
| `icon_dark.png` | Icon for dark mode |
| `icon_dark@2x.png` | High-DPI icon for dark mode |
| `logo_dark.png` | Logo for dark mode |
| `logo_dark@2x.png` | High-DPI logo for dark mode |

## Adding your images

Replace the placeholder files in this directory with actual Pool Lab branding.
Use a clean, recognizable representation of the Pool Lab device or logo.
Ensure transparent backgrounds for proper rendering in both light and dark themes.
